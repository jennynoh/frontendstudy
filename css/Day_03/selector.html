CSS 선택자

요소에 서식이나 배경을 적용하기 위해서는 적용하려는 요소를 정확히 선택할 수 있어야 한다.
선택자는 이러한 요소들을 다양한 방식으로 선택할 수 있도록 해준다.

  • 단순 선택자
    • 전체 선택자 (*): 전체 요소를 선택할 때 사용 (특이값: 0)
    • 태그명 선택자: 태그이름으로 선택할 때 사용 (특이값: 1)
    • 그룹 선택자: 콤마(,)를 이용하여 동일한 스타일 정의를 가진 모든 HTML 요소를 선택할 때 사용 (특이값: )
    • 클래스 선택자: class 속성으로 지정한 값을 가진 모든 요소들을 선택할 때 사용 (클래스는 같은 값을 여러 요소에 사용가능하므로 복수 선택 가능) (특이값: 10)
    • id 선택자: id 속성으로 지정한 값을 가진 요소를 선택할 때 사용 (id는 하나의 요소에만 사용하므로 단일요소 선택할 때 사용) (특이값: 100)
    • (예) 단순 선택자의 사용 예
  • 가상(Pseudo) 클래스 선택자
    • 참고: w3schools 가상클래스
    • 가상클래스란 태그나 특정 다른 선택자 뒤에 콜론(:)을 붙여 추가기능을 부여하는 키워드입니다. 선택한 요소가 특별한 상태여야 만족되는 클래스입니다.
    • 주로 a태그의 상태를 나눠서 가상클래스를 적용했으나, 다른 태그에도 사용할 수 있습니다.
    • (예) 가상클래스 선택자 사용 예
      (연습) 테이블에 가상 클래스 선택자 사용
  • 가상(Pseudo) 요소 선택자
    • 참고: w3schools - 가상요소
    • 가상요소는 태그의 지정된 부분에 스타일을 적용할 때 사용합니다.
      콜론 두개 (::)를 붙여 요소의 일부분에 스타일을 적용할 수 있습니다.
      예를 들어 요소의 첫글자, 요소의 첫줄, 콘텐츠 앞이나 뒤에 또다른 콘텐츠 삽입 등을 할 수 있습니다.
    • (예) 가상요소 선택자 사용 예
  • 속성 선택자
    • 특정 속성이나 속성값을 가진 html 요소를 선택할 때 사용합니다.
    • [속성]
      [속성=값]: 속성이 값인..
      [속성~=값]: (띄어쓰기로 구분된 단어) 값을 포함하는..
      [속성|=값]: (띄어쓰기, -으로 구분된) 값을 포함하는..
      [속성^=값]: 속성이 값으로 시작하는..
      [속성$=값]: 속성이 값으로 끝나는..
      [속성*=값]: 속성이 값을 포함하는..
    • 참고: w3schools - 속성선택자
    • (예) 속성 선택자의 사용 예
  • 종속 선택자
    • 둘 이상의 선택자를 조합해서 사용하는 것
    • 공백 (후손 선택자): 하위항목 선택
    • > (자식 선택자): 인접한 하위항목 선택
    • + (형제 선택자): 같은 상위 클래스를 가진 연달아 나오는 다음 요소 선택
    • ~ (일반 형제 선택자): 같은 상위 클래스를 가진 모든 요소들 선택
    • (예) 종속 선택자의 사용 예
  • 기타
    • not(선택자): 괄호안에 지정한 선택자가 아닌 요소 선택
    • nth-child(수식): 수식에 맞는 요소 선택
    • (예) 기타 선택자
