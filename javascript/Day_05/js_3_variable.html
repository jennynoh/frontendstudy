변수의 선언과 Scope

결과 at console

자바스크립드는 weakly typed 언어입니다.
Weakly typed 언어란 변수를 선언한 후 다른 타입의 데이터를 대입할 수 있는 언어입니다.
반면 자바는 strongly typed 언어로 선언된 변수엔 동 타입의 데이터만 대입할 수 있습니다.

Javascript Scope의 개념: local, global

  1. 지역공간에서는 글로벌 공간에 접근 가능, 반대는 불가능
  2. 지역변수는 전역변수보다 우선순위가 높다.
  3. arrow function (화살표함수): scope의 개념이 없음
