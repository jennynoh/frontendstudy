영역 분할 태그

브라우저 내에서 공통된 역할을 담당하는 요소들을 집합시켜 다른 요소와 구별하고
그룹화된 요소를 브라우저의 특정 위치에 배치하기 위한 태그입니다.

div 태그 - 영역분할에 사용되는 대표적인 태그

블럭 요소의 특징
  • 위에서 아래로 쌓이는 구조
  • 경계(=border)가 있음. 속성은 border-box
  • 너비와 높이가 있음 - css를 이용해 너비, 높이, 위치 조정 가능
  • div 태그 외에도 sementic 태그인 header, footer, section, article, hn, p, ol, ul, dl 등도 블럭요소로 너비와 높이가 있음
    (인라인 요소는 너비와 높이를 조정할 수 없음 - 속성이 없음)
  • 인라인 요소를 블럭요소로, 블럭요소를 인라인 요소로 변경할 수 있음

span 태그 - 인라인 요소이면서 인라인 요소의 영역분할 담당

인라인 요소의 특징
  • 왼쪽에서 오른쪽으로 쌓이는 구조
  • 경계 속성이 없음 (border-box x)
  • 너비와 높이를 지정할 수 없음
  • b, i, u, img, a, span, (텍스트) ...
