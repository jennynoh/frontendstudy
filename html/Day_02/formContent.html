사용자의 정보를 입력받는 태그

form 요소는 사용자의 데이터를 입력받는 요소들을 의미합니다.
form 요소의 대부분은 인라인 요소입니다.
회원가입 화면, 로그인 화면, 회원정보 수정화면, 검색창 등이 모두 form 관련 태그로 구성됩니다.

input 태그 중 입력상자: type="text", type="password", type="hidden"

사용자 입력

type="radio" : 여러 항목 중 한가지만 선택 (Radio Button)

Location 더큰집 광교 앨리웨이 인천국제공항 트레이드타워

select / option : 여러 항목 중 하나 선택

결제방식

type="checkbox" : 여러 항목 중 여러개 선택

추가메뉴(옵션) 감자튀김   주먹밥   고구마튀김   날치알밥   쿨피스  

select / option : multiple 속성을 이용해서 여러 개 선택 가능

textarea : 여러 줄 글 상자

    input type의 속성: input 태그의 성격을 정의한다.

  • type="text" : 한 줄짜리 텍스트를 입력하기 위한 용도
  • type="password" : 비밀번호 등 외부에 노출되면 안되는 정보를 입력하기 위한 용도
  • type="hidden" : 사용자에게 보이지 않지만 서버로 데이터를 전송시키는 필드 (*value, name 지정 필수)
  • type="button" : 버튼 모양을 생성. javascript와 연동하여 기능 붙임
  • type="submit" : form action 속성에 지정한 URL(or 서버 프로그램 경로)로 데이터 전송
  • type="reset" : 입력필드에 입력한 값을 초기화
  • type="file" : 파일을 서버에 첨부해서 전송하기 위한 버튼 (*form 태그 method="post", multipart/form-data 속성 추가 필수)
