<a> 태그 - 링크를 표기하는 태그

링크를 만드는 태그 <a> 태그는 클릭하면 연결된 곳으로 이동하는 태그입니다.
텍스트를 안에 넣으면 해당 텍스트가 웹 상에 표시되고, 클릭했을 때 삽입된 링크로 이동합니다.
텍스트 대신 이미지를 넣어 링크를 삽입할 수도 있습니다.

a 태그의 중요 속성

  • href: 링크를 생성해서 이동할 목적지의 파일이나 웹 문서상의 다른 주소
  • target: 링크의 내용이 표시될 위치
    • _self: 현재 창에 바로 열림 (default)
    • _blank: 새로운 창에 열림
    • 사용자 지정 이름: 사용자가 지정한 이름의 창으로 열리고, 없으면 새 창으로 열림

  • 구글
  • 다음

이미지 링크


  • 염라를 위한 안내

  • 경이로운 소문

문서 내 특정 위치로 이동

href 속성값으로 특정 문서의 위치로 이동하기 위해서는 글로벌 속성인 id를 지정하고
해당 아이디가 적용된 요소를 <a>태그의 href로 지정하면 클릭 시 해당 아이디의 위치를 브라우저 상단에 놓습니다.
