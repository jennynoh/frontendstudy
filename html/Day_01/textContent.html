텍스트 컨텐츠를 표기하는 태그

html 문서의 body 태그에는 기본적인 데이터를 표현하기 위해
각 데이터를 표기하는 태그를 사용해야한다.
텍스트를 표현하기 위해 사용하는 태그들은 어떤 것들이 있을까?

제목태그

hn 태그 - Heading

hn 태그는 제목을 표기하는 태그입니다.
제목의 용도에 따라 1~6 단계로 구분됩니다.

h1: 로고, 표지제목

h2: 대제목, 단락제목

h3: 중제목, 페이지 제목

h4: 소제목, 문단제목

h5: 문단제목
h6: 주석제목

본문태그, 문단태그

p 태그 - Paragraph

p 태그는 문단이나 본문을 표기하는데 사용되는 태그입니다.
작성자가 여러 개의 문단으로 구성하고자 하면 여러 개의 p태그를 사용하여 표기합니다.

줄바꿈 태그

br 태그 - Break Line

br 태그는 한 줄의 문장을 둘로 나눌 때 사용하는 태그입니다.
여러 개의 br 태그를 사용하면 줄 사이에 여백이 생깁니다.
따라서 여러 줄을 나누기 위해 br태그를 반복적으로 사용하는 일은 없어야합니다.

수평선 태그

hr 태그 - Horizobtal Rule

hr 태그는 한 줄의 수평선을 만들어주는 태그입니다.
속성값(CSS 등)을 통해 선의 모양을 바꿀 수 있다.

텍스트를 한 줄로 표기하면서 강조하는 태그

Strong 태그 - 굵게 강조할 텍스트
b 태그 - 굷게 강조할 텍스트
i 태그 - 이텔릭체로 표기하는 태그
em 태그 - 이탤릭체로 표기하는 태그
span 태그 - 줄 나눔 없이 영역을 묶기 일부 글자만 스타일을 적용하고자 할 때 시용합니다.
u 태그 - 링크를 표시하기 위한 용도가 아니라 단순 밑줄을 표시하는 태그
